Pour pouvoir utiliser cette application en local : 

0.  Télécharger le dossier complet.
1.	Avec l’Invite de Commande (ou Terminal), allez dans le dossier StresssApp (ex. : cd C:\Users\StresssApp )
2.  Taper la commande "npm install" pour vérifier et installer tous les modules nécessaires 
3.	Tapez la commande « iconic serve » pour lancer le serveur ionic depuis le dossier de l’application, normalement votre navigateur par défaut s’est ouvert avec la page d’acceuil de l’app
4.  Jouer autant que vous le souhaitez !
